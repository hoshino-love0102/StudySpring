# StudySpring

SpringBoot 스터디 기록 저장소입니다.

## 프로젝트 목록
- `loginproject`: 로그인 기능 실습
- `NoteProject`: 노트/ToDo 리스트 기능 구현 (JavaScript + Spring 연동)
- `lostfound_project`: 로그인 기능 구현, 게시글 저장 & 삭제 기능 구현, 날짜 db에 저장 구현

## 사용 기술
- Java 17
- Spring Boot
- HTML/CSS/JavaScript
- Gradle

## 업데이트 로그
- 로그인 기능 기본 구조 구성
- 노트 저장, 삭제 기능 개발
- 비밀번호 암호화 기능 구현
- LocalDateTime기반 날짜 저장 구현
- 계속 업데이트 예정
